# Task: Plan View Layer (Cloudflare & Server)

High-availability Cloudflare system for serving videos and the HTML pages which render them as performantly and efficiently as possible. We care about this because:

1. Better UX & Speed for viewers.
2. Higher availability than we can manage with just our Hertzner VPS.
3. Handles spikes in traffic to videos.
4. Can continue to serve content if the Hertzner boz is temporarilly down for any reason.
